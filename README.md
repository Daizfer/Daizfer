## 🛡️ Proyecto intermodular – Cloud Honeypot Lab (OCI ARM64)

Diseño e implementación de una infraestructura segmentada en Oracle Cloud (Always Free – ARM64), orientada al análisis controlado de tráfico y eventos de seguridad.

Arquitectura compuesta por:

- 🔹 Honeypots SSH y Web (Cowrie y DVWA) expuestos de forma controlada
- 🔹 VPN WireGuard como único punto de acceso administrativo
- 🔹 SIEM (Wazuh) en red privada para monitorización y correlación de eventos
- 🔹 Envío de logs desde zona pública a entorno aislado
- 🔹 Segmentación Public / Private aplicando principios básicos de Zero Trust
- 🔹 Medidas de hardening (cambio de puerto SSH, control de exposición y reglas de red)

El objetivo del proyecto es simular una infraestructura corporativa segura, analizar intentos de intrusión reales y documentar vectores de ataque en un entorno aislado y reproducible.
